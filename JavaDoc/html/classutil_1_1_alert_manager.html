Detailed Description

Classe che gestisce gli alert in modo centralizzato all'interno di tutta l'applicazione.

La classe fornisce un metodo per mostrare alert di tipo diverso (di errore, di conferma) in modo centralizzato. Gestisce il titolo, il contenuto e la tipologia di alert.

ButtonType util.AlertManager.showAlert ( String title,
String message,
String type )

Mostra un alert di tipo specificato.

Il metodo crea un alert con il tipo e il titolo specificato. Nel caso in cui l'alert sia di tipo 'CONFIRMATION', l'utente dovrà rispondere tramite un pulsante ('ok' o 'confirm'); tutti gli altri tipi di alert vengono mostrati senza aspettarsi una risposta

Parameters
titleTitolo dell'alert
messageMessaggio dell'alert
typeTipo dell'alert (es. "CONFIRMATION", "ERROR", ecc.)
Returns
La risposta dell'utente sotto forma di ButtonType, se il tipo 'CONFIRMATION'

Restituisce ButtonType#CANCEL se l'utente non seleziona nulla. Se il tipo non è "CONFIRMATION", non restituisce nulla.

Precondition
type deve essere un valore valido riconosciuto dalla classe Alert.AlertType.
Postcondition
Mostra un alert all'utente e restituisce la risposta dell'utente (solo per gli alert di tipo "CONFIRMATION").
