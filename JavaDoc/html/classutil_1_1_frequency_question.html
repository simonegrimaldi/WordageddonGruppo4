Detailed Description

Domanda del quiz che chiede "Quante volte compare una certa parola in un documento?".

Questa classe genera una domanda di tipo frequenza, in cui viene chiesto quante volte una parola specifica appare in un dato documento. La risposta corretta è la frequenza di quella parola, mentre le opzioni di risposta includono la risposta corretta più 3 risposte plausibili ma errate.

util.FrequencyQuestion.FrequencyQuestion ( List< AnalysisImpl > analyses)

Costruisce una domanda di frequenza basata su un'analisi.

Questo costruttore seleziona una parola casuale dal documento analizzato, quindi costruisce una domanda chiedendo quante volte questa parola appare nel documento. La risposta corretta è la frequenza di quella parola nel testo. Inoltre, vengono generate 3 opzioni errate ma plausibili.

Parameters
analysisL'analisi del documento da cui verrà estratta la parola e la frequenza.

int util.FrequencyQuestion.valuta ( String s)

Valuta la risposta dell'utente.

Questo metodo confronta la risposta selezionata dall'utente con la risposta corretta. Se la risposta è corretta, l'utente guadagna 10 punti, altrimenti perde 3 punti.

Parameters
sLa risposta selezionata dall'utente come stringa.
Returns
Il punteggio dell'utente (10 per risposta corretta, -3 per risposta errata, 0 se la risposta è nulla).
Postcondition
Restituisce il punteggio in base alla risposta dell'utente.
