Detailed Description

Classe di gestione della connessione al database PostgreSQL.

La classe si occupa di gestire la connessione al database. Fornisce un metodo statico per ottenere la connessione al database e garantisce che venga creata una sola connessione riutilizzabile durante l'intero ciclo di vita dell'applicazione, in questo modo evitiamo la creazione di connessioni multiple, migliorando quindi l'efficienza dell'applicazione.
