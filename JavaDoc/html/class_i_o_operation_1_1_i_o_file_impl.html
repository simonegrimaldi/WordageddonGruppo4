Detailed Description

Implementazione concreta per l'interfaccia 'IOFile' per il salvataggio e il caricamento dei file.

La classe fornisce le operazioni concrete per il contenuto di un file di testo e per salvare un file in una destinazione specificata.

◆ loadFile()

String IOOperation.IOFileImpl.loadFile ( List< String > filePaths)

Carica il contenuto di uno o più file di testo.

Il metodo apre ogni file presente nell'elenco 'filePaths', legge il contenuto di ogni file e restituisce il risultato complessivo concatenato in una stringa.

Nel caso in cui uno dei file non viene trovato, verrà sollevata un eccezione e stampato un messaggio che ci avvisa che il file non è stato trovato.

Parameters
filenamelista dei file da caricare
Returns
Il contenuto del file selezionato sotto forma di unica Stringa

boolean IOOperation.IOFileImpl.saveFile ( File destFile,
File selectedFile )

Salva il contenuto di un file selezionato in uno specifico file di destinazione.

In particolare, il metodo copia il contenuto del 'selectedFile' nel file di destinazione 'destFile'. Nel caso in cui il file di destinazione esista già, allora questo verrà sovrascritto.

Parameters

Returns
