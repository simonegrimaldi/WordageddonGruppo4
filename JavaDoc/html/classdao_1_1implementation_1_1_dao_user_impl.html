Detailed Description

Implementazione dell'interfaccia DaoUser per la gestione delle operazioni sugli utenti.

La classe fornisce i metodi per gestire l'autenticazione e la registrazione degli utenti nel database.

La classe utilizza il Data Acces Object pattern per separare la logica di accesso ai dati dalla logica applicativa.

I metodi presenti nella classe permettono di verificare le credenziali dell'utente, di registrare nuovi utenti e di gestire il recupero e l'archiviazione delle informazioni utente nel database.
